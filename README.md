# Vanilla JS ES6 / Webpack Starter Project

This repo is a boilerplate for starting a Vanilla JS project.

## Support

- JavaScript ES6
- SCSS
- Jest

## Installation

Just download it and run:

```bash
npm install
```
then

```bash
npm start
```

To build the bundle for production

```bash
npm run build
```

For running unit test

```bash
npm run test
```
